# Transformer

A little app that does data transformations.

## Setup

```
virtualenv --python=python2.7 env27
. env27/bin/activate
pip install -r requirements.txt
```

## Run Locally

```
. env27/bin/activate
DEBUG=true PORT=8080 python transformer/app.py
```

## Test Locally

```
. env27/bin/activate
./run_tests.sh
```
